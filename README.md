## Setup virtualenv

### Install virtualenvwrapper

    sudo pip install virtualenvwrapper

### Create env
We are using Python 3, since we are not in the stoned age.

    mkvirtualenv -p /path/to/python3 envname

### Install requirements
While in enviroment:

    pip install -r req.txt
